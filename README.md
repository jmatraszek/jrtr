# JRTR

JRTR


## Usage

```
$ jrtr --help`
jrtr 0.1.0
Jakub Matraszek <jakub.matraszek@gmail.com>
Render Handlebars templates with data from context json file.

USAGE:
    jrtr [OPTIONS] --context <context> --template <template>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -c, --context <context>      Input context file
    -o, --output <output>        Output file, stdout if not present
    -t, --template <template>    Input template file
```
